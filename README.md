# TODO List

Web Application em ruby on rails para gerenciamento de tarefas.

## Requerimentos

### Versões utilizadas :

>  Ruby : version 2.3.1 

>  Rails : version 4.2.6

>  Sqlite3 : 3.11.0

>  Bundler : 1.17.3

## Instalação

```
$ git clone

$ bundle install
```

## Criação e inicialização banco de dados

```
$ rake db:migrate
```


## Rodando testes

```
$ bundle exec rspec
```
